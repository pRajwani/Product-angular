Managing Products

You can perform some actions on your Products List to Create, Edit, Delete, Sort and Filter Products. Below are the list of actions -
add    

Add - To create a new product. Clicking this button will create a new row where you can fill detail of new product.

create    

Create - To edit a product. Clicking this button will make the row editable where you can edit detail of product and choose new image of the product.

delete    

Delete - To delete a product. Clicking this button will delete the corresponding product. You confirmation alert, don't worry if you click that button accidentally.

check    

Confirm - To confirm changes. Clicking this button will confirm the changes you made in the editable fields while entering or editing product details.

clear    

Cancel - To cancel the current operation. Clicking this button will cancel the operation you were just performing like creating a new product or editing the details of the product.

Sort - You can sort the data in the Alphabetical or Numerical order by clicking the column title. It can be in ascending or descending.

Filter - You can filter the data by using the input fields respective to their columns. It can be handy if have lots of products. Don't mistook with creating the new product. Creating new product has a seperate button.
